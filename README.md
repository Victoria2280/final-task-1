# Final Task 1 — Console Monster Fighting Game

A console-based game developed as the first final project for the Programming Course at KIT (Karlsruhe Institute of Technology).

## About

The player fights against monsters in a turn-based console game. The game is implemented in Java and runs entirely in the terminal.

## Technologies

- Java
- IntelliJ IDEA

## How to Run

1. Clone the repository:
```bash
   git clone https://github.com/mohashka-code/final-task-1.git
```
2. Open the project in IntelliJ IDEA
3. Run the main class from `src/edu/kit/kastel/`
